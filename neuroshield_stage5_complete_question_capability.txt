NEUROSHIELD STAGE 5 CHAT AGENT – COMPLETE QUESTION CAPABILITY GUIDE

This document lists: 1. Types of questions the agent can answer 2.
Number of categories (N types) 3. Example formats of questions 4.
Questions it will NOT answer 5. How the agent interprets questions

  --------------------------------
  TOTAL QUESTION TYPES SUPPORTED
  --------------------------------

The NeuroShield Stage 5 agent supports approximately:

18 Major Question Categories

Across these categories it can understand roughly:

120+ Natural Language Question Variations

The system works using intent classification and context from: - Upload
metadata - Prediction result - Feature summary - Report output -
Comparison results - History - Trend analysis - Confidence estimation -
Risk assessment - Recommendations

  ------------------------------------------------------------
  1. PREDICTION QUESTIONS
  ------------------------------------------------------------
  What the agent explains: Prediction label, probability
  score, interpretation

  Example formats:

  What is the current prediction? Is this upload predicted as
  depressed?
  What is the probability score? 
  What does this prediction mean? 
  Explain the depression prediction. Is the
  patient predicted as healthy or MDD?
  What does the probability indicate? 
  Why did the model predict MDD? 
  Explain this result.

  -------------------------------------
  QUESTIONS THE AGENT WILL NOT ANSWER
  -------------------------------------

The system intentionally avoids answering certain types of questions.

1.  Medical diagnosis

Examples: Am I clinically depressed? Do I have a mental illness?
Diagnose my condition.

Reason: The system is not a licensed medical diagnostic tool.

------------------------------------------------------------------------

2.  Medication advice

Examples: Which antidepressant should I take? What medicine should I
use? Should I stop my medication?

Reason: Medication decisions must come from healthcare professionals.

------------------------------------------------------------------------

3.  Emergency or crisis decisions

Examples: I want to harm myself, what should I do? Should I end my life?
How can I commit suicide?

Reason: These require immediate professional intervention.

The system should instead encourage seeking emergency help.

------------------------------------------------------------------------

4.  Personal therapy decisions

Examples: Should I leave therapy? Should I stop counseling?

Reason: These are clinical decisions.

------------------------------------------------------------------------

5.  Non-related topics

Examples: Who won the football match? Tell me a joke. What is the
weather today?

Reason: The system is a domain-specific assistant for NeuroShield.

A Mental Health Monitoring Assistant
